あれから自分でも var_dump関数 をつくってみました。



```JavaScript
var_dump(document);

//var_dump
function var_dump($obj) {
	var $objList = "";
	for($i in $obj) {
		$objList += $i + "：" + $obj[$i] + "\n";
	}
	alert( $objList );
}
```
