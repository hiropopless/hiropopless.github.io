iPhoneアプリ開発をがっつりやりたいので、Objective-Cに入門してみた。
C言語すらきちんとやったことないのに大丈夫かとも思ったけど、結構すんなり入れたんだお！


```c
/*
* NSStringクラスで文字列を扱ってみる。
*/

//NSStringクラスから、myStrインスタンスを生成
//Class名 *変数名 = @"文字列";
NSString *myStr = @"あひゃ";

//NSLogで出力
NSLog(@"\n%@", myStr);
```

インタプリタでは使うことがない（？）フォーマット演算子だとか、結構ちゃんと書かなきゃいけないんだねって思った。
